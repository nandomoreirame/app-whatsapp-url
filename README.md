# [WhatsApp URL](https://whatsapp-url.netlify.com/)

> App Whatsapp URL - With Vuejs

## Install

```bash
git clone https://github.com/nandomoreirame/app-whatsapp-url.git
cd app-whatsapp-url/
yarn install
yarn serve
```

open [localhost:8080](http://localhost:8080/)
